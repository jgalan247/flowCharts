Code → Flowchart: Iteration

Produce the equivalent flowchart for a Python snippet containing a loop

Task: Study the Python code below carefully. Then answer the comprehension questions to confirm your understanding, and use the canvas to draw the equivalent flowchart. Note how the loop structure maps to a decision diamond and a back-edge (arrow looping back).
Python
# Sum integers from 1 to a user-supplied limit
total = 0
counter = 1
limit = int(input("Enter limit: "))

while counter <= limit:
    total = total + counter
    counter = counter + 1

print("Sum =", total)

Comprehension questions — answer before you draw

Q1. What shape is used to represent the while counter <= limit condition in a flowchart?

Q2. How many process boxes are needed inside the loop body?

Q3. In the flowchart, the "YES" branch of the counter <= limit decision should point…

Q4. After the last iteration, which flowchart element comes immediately after the decision's "NO" branch?
